## 📝 Store Listing

### Short Description (80 characters max)
"Discover landmarks from your airplane window with AI-powered stories"

### Full Description (4000 characters max)

**See the World from Above with SkyLens**

Transform every flight into an adventure! SkyLens is your intelligent window companion that identifies landmarks, mountains, cities, and natural wonders as you fly overhead.

**✈️ Key Features:**

🗺️ **Real-Time Landmark Detection**
- Automatically identifies mountains, cities, monuments, and natural wonders along your flight path
- Works with both real GPS tracking and mock flight simulations
- Shows landmarks within 200km of your route

📸 **Beautiful Photos**
- High-quality images from Wikimedia Commons, Pixabay, and Pexels
- Offline photo download for use without internet
- Photo carousel with multiple views of each landmark

🤖 **AI-Powered Stories**
- Fascinating stories about each landmark generated by Claude AI and Google Gemini
- Real-time flight narration that describes what you're seeing below
- Ask AI questions about landmarks and geography

🔔 **Smart Predictions**
- Get notified when landmarks are about to become visible
- "Upcoming" alerts 5-10 minutes before landmarks appear
- Velocity-based predictions that adapt to your flight speed

✈️ **Flight Planning & Tracking**
- Search 7,000+ airports worldwide
- Plan routes between any two airports
- Track real flights with GPS or simulate routes for testing
- Great circle route visualization on interactive maps

📊 **Trip History & Replay**
- Save all your flights with visited landmarks
- Replay past trips and relive the journey
- Export trips as GeoJSON files to share

🌍 **Offline First**
- Download offline packs for routes (landmarks + photos + facts)
- Background downloads with progress notifications
- Works without internet connection after download

⚙️ **Customizable**
- Toggle between metric and imperial units
- Choose AI provider (Claude or Gemini)
- Dual GPS mode: Real tracking or Mock simulation

**Perfect for:**
- ✈️ Frequent fliers who want to learn about the world
- 🗺️ Geography enthusiasts and travelers
- 📚 Students learning about world landmarks
- 📸 Photography lovers collecting views from above

**Database:**
- 138 hand-curated landmarks (mountains, cities, monuments, volcanoes, rivers, lakes, glaciers, deserts, canyons, islands)
- 7,000+ airports worldwide
- Expandable with community contributions

**Privacy:**
- No account required (optional Google Sign-In for cloud sync)
- Offline mode available
- Location data never leaves your device
- Open-source friendly architecture

Download SkyLens today and never miss a landmark on your next flight!

## 📊 Data Safety

### Data Collection

**Location Data:**
- ✅ Collected: Yes
- ✅ Purpose: App functionality (flight tracking)
- ✅ Shared: No
- ✅ Optional: Yes (can use offline mode)
- ✅ Encrypted: Yes (in transit)

**User Account Data:**
- ✅ Collected: Optional (email via Google Sign-In)
- ✅ Purpose: Cloud sync (future feature)
- ✅ Shared: No
- ✅ Optional: Yes (can skip sign-in)
- ✅ Deletable: Yes (account deletion in settings)

**App Activity Data:**
- ✅ Collected: Yes (trip history)
- ✅ Purpose: App functionality
- ✅ Shared: No
- ✅ Optional: No (core feature)
- ✅ Encrypted: Yes
- ✅ Deletable: Yes (can delete trips)

**Photos:**
- ❌ Not collected from user
- ✅ Downloaded from public APIs only

### Security Practices

- ✅ Data encrypted in transit (HTTPS)
- ✅ Data stored locally in Room database
- ✅ Users can request data deletion
- ✅ No third-party data sharing
- ✅ No advertising or tracking SDKs

## 🔧 Build Commands

### Build Release AAB
```bash
cd android
./gradlew bundleRelease
```
Output: `app/build/outputs/bundle/release/app-release.aab`

### Build Release APK (for testing)
```bash
./gradlew assembleRelease
```
Output: `app/build/outputs/apk/release/app-release.apk`

### Test Release Build
```bash
./gradlew installRelease
```
